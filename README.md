# 🚗 WiFi-Controlled Camera Rover (ESP32 + Raspberry Pi)

A WiFi-controlled robotics rover built using an ESP32-based chassis and a Raspberry Pi, featuring real-time video streaming, object tracking, and remote control via WebRTC.

## ⚙️ Features

- Remote control over WiFi
- Real-time video streaming (WebRTC)
- Object tracking (OpenCV-based)
- Modular Python control architecture
- ESP32 motor control interface
- Browser-based dashboard (WebRTC + data channel)

---

## 🧠 Tech Stack

- ESP32 (motor control + telemetry)
- Raspberry Pi (vision + control)
- Python (control + processing)
- OpenCV (tracking)
- WebRTC / Janus (streaming + control)
- UDP messaging between modules

---

## 🚀 Project Goals

- Build a fully controllable rover from scratch
- Integrate real-time video + control
- Explore tracking and autonomous behaviours
- Develop a foundation for SLAM and navigation
